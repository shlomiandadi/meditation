# OriOnePage — דף נחיתה מדיטציות

דף נחיתה מכירתי בעברית (RTL) לסט 14 מדיטציות של מאור אילוז.

## הפעלה

```bash
npm install
npm run dev
```

פתח [http://localhost:3000](http://localhost:3000)

## בנייה לפרודקשן

```bash
npm run build
npm start
```

## התאמות

- **קישורי תשלום**: עדכן ב-`src/lib/checkout.ts` (גברים / נשים — Schooler).
- **תמונת מאור**: הוסף תמונה ב-`public/maor.jpg` והצג בקטע המרצה.
- **משך מדיטציות**: עדכן ב-FAQ ב-`page.tsx`.
